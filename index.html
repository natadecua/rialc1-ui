About the Model & Potree Integration

This UI visualizes a tree species identification model using Potree for high-performance LiDAR point cloud visualization. Potree provides industry-standard tools for forestry analysis including classification-based coloring, measurement tools, and clipping volumes. Tree detection results are displayed as 3D markers directly on the source point cloud data.

🔧 Potree Workflow

  1. Inspect & Prepare: Use QGIS to load LAS files and shapefiles, verify coordinate systems
  2. Convert: Use PotreeConverter to convert LAS files to Potree format
  3. Visualize: Load converted data in this Potree-based viewer for analysis
  4. Analyze: Use measurement tools, clipping volumes, and tree markers for detailed examination

Potree Setup Instructions

🚨 For Best Results - Use PotreeConverter

For optimal performance with large LAS files, convert them using PotreeConverter first:

Download PotreeConverter:
https://github.com/potree/PotreeConverter/releases
Convert LAS to Potree format:
./PotreeConverter.exe input.las -o output_directory
Then load the converted data:
Use "Load Potree Point Cloud" button above
✅ Alternative: Direct File Loading

You can also load LAS files directly, but performance may be limited for very large files.

Local Server (PowerShell) - if needed:
cd "path/to/rialc1-ui"
python -m http.server 8000
Then visit:
http://localhost:8000
